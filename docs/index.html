🕸️ Domínio Web (Thonsons Routers)

Guia completo de instalação para Ubuntu 24.04, 25.04 e 26.04 LTS

📌 Sobre

Este guia ensina a instalar o Domínio Web Plugin (Thonsons Routers) em distribuições Ubuntu modernas (24.04+), fazendo o AppController e o TRComputerPluginLinux funcionarem corretamente.

📥 Download dos Arquivos

ArquivoTamanhoOrigem
gg-client-linux.deb~10 MBCentral de Soluções
DominioWebPlugin.7z~22 MBCentral de Soluções

💡 Coloque os dois arquivos em ~/Downloads/ antes de executar o script.

🚀 Instalação Automática

Baixe e execute o script de instalação:

wget -O install-dominio-web.sh https://raw.githubusercontent.com/JosueMadureira/dominio-web-plugin-linux/main/install.sh
chmod +x install-dominio-web.sh
./install-dominio-web.sh

🛡️ O script resolve todas as dependências automaticamente.
